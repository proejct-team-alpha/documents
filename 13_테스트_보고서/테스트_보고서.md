### 5.1 예약 라이프사이클

```
RESERVED -> RECEIVED -> IN_TREATMENT -> COMPLETED
    |           |
    v           v (접수 취소: RESERVED로 롤백)
 CANCELLED   CANCELLED (cancelFully: 즉시 취소)
```

- 모든 상태 전이 경로에 대한 테스트 존재
- 불가능한 전이(COMPLETED/IN_TREATMENT -> CANCELLED) 방어 검증

- 퇴직일(날짜+시간) 조합 및 정합성 검증 (한쪽만 입력 시 예외)
- 과거 퇴직일 입력 시 즉시 비활성화
- 스케줄러를 통한 만료 직원 자동 비활성화

### 5.3 LLM 증상 분석

- LLM 응답 파싱 (정규식 패턴, 대괄호, 텍스트 직접 탐색)
